# Space Tourism Front

Frontend for the Space Tourism project. Built with React, Vite, Tailwind CSS, shadcn/ui, React Router, Zustand, and TanStack Query.

## Stack

- **React** (latest) + TypeScript
- **Vite** – build tool
- **Tailwind CSS** – styling
- **shadcn/ui** – UI components (new-york style, zinc base)
- **React Router** – routing
- **Zustand** – client state
- **TanStack Query** – server state / data fetching

## Setup

```bash
pnpm install
```

## Run

```bash
pnpm dev
```

Open [http://localhost:5173](http://localhost:5173).

## Scripts

- `pnpm dev` – start dev server
- `pnpm build` – production build
- `pnpm preview` – preview production build
- `pnpm lint` – run ESLint

## Adding shadcn/ui components

```bash
pnpm dlx shadcn@latest add <component-name>
```

Example: `pnpm dlx shadcn@latest add card`
